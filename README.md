## Ghost Hello World
A hello world function for learning and testing purposes.

#### Usage
```ts
import { helloWorld } from 'ghost-hello-world';

helloWorld({firstName: 'Rahul'});
helloWorld({firstName: 'Rahul', lastName: 'Ghost'});

// Output:
// Hello Rahul!
// Hello Rahul Ghost!
```
